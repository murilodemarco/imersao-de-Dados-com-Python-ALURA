# 📊 Dashboard de Salários na Área de Dados

## 🚀 Sobre o Projeto

O **Dashboard de Salários na Área de Dados** foi desenvolvido durante a **Imersão de Dados com Python da Alura**, com o objetivo de transformar dados em insights visuais de forma prática e interativa.

A aplicação permite analisar informações salariais da área de tecnologia e dados através de filtros dinâmicos e gráficos interativos, proporcionando uma experiência intuitiva para exploração dos dados.

O projeto demonstra habilidades importantes para a área de desenvolvimento e análise de dados, como:

* ✅ Manipulação e análise de dados com Python
* ✅ Criação de dashboards interativos
* ✅ Desenvolvimento com Streamlit
* ✅ Visualização de dados com Plotly
* ✅ Organização e estruturação de projetos
* ✅ Boas práticas de programação

Além da parte técnica, o projeto também evidencia preocupação com experiência do usuário, organização visual e clareza na apresentação das informações.

## ⚙️ Como Executar o Projeto

### 🔧 Pré-requisitos

Antes de começar, você precisará ter instalado:

* Python 3.10+
* Pip

---

### ▶️ Executando localmente

```bash
# Clone o repositório

# Acesse a pasta do projeto

# Instale as dependências
 pip install -r requirements.txt

# Execute a aplicação
 streamlit run app.py
```

Após executar, o Streamlit abrirá automaticamente no navegador.
